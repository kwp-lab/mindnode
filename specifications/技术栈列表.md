## UI Components (UI 组件)

MindNode AI use [Radix UI](https://www.radix-ui.com/primitives/docs/overview/introduction) and [shadcn/ui](https://ui.shadcn.com/) for UI components.

Radix UI 是一个无样式（headless）的 React 组件库，专注于提供功能完善且符合无障碍标准（WAI-ARIA）的组件逻辑。它不包含任何样式，旨在让开发者根据自己的设计系统进行样式定制。

shadcn/ui 则是在 Radix UI 的基础上，结合 Tailwind CSS，提供了一套带有默认样式的可复用组件集合。它通过 CLI 工具将组件代码直接复制到项目中，允许开发者完全控制和定制这些组件。

## CSS Framework

[tailwindcss](https://tailwindcss.com/) v4.1 - A utility-first CSS framework packed with classes like flex, pt-4, text-center and rotate-90 that can be composed to build any design, directly in your markup.

## Icon Library

[Lucide](https://lucide.dev/) is an open-source, community-driven icon library offering clean, consistent, and customizable SVG icons for modern web and app interfaces.

## React Flow

[React Flow](https://reactflow.dev/) is a React library for building interactive, customizable node-based diagrams and flowcharts with drag-and-drop functionality.

## Next.js & Next.js App Router

Next.js is a React-based full-stack framework for building fast, SEO-friendly web applications with built-in routing, rendering, and backend capabilities.

[Next.js](https://nextjs.org/) App Router is the modern routing system in Next.js that uses React Server Components and layouts to build scalable, performant full-stack applications with a component-centric architecture.

## AI SDK by Vercel

The [AI SDK](https://ai-sdk.dev/docs/introduction) is the TypeScript toolkit designed to help developers build AI-powered applications and agents with React, Next.js, Vue, Svelte, Node.js, and more.

### AI Elements

[AI Elements](https://ai-sdk.dev/elements) is a component library and custom registry built on top of shadcn/ui to help you build AI-native applications faster. It provides pre-built components like conversations, messages and more.
